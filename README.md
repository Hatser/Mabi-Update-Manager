# Mabi-Update-Manager
Mabinogi update manager which supports multithreading, implemented in python (only works on KR server)

## Usage
```python
import MabiUpdate
MabiUpdate.MabinogiEnvironment('C:\Your\Mabinogi\Path', 'http://patchtext.mabinogi.com/patch/patch_kor.txt')
```
## Requirements
numpy
tqdm
